Рекламные конструкции. Город Пенза

Аналитический проект на открытых данных.
Заказчик: Самостоятельный проект.
Цель: Найти полезные рекомендации для рекламодателей.
Описание проекта: Этот проект придуман самостоятельно и сделан с использованием открытых данных для того, чтобы продемонстрировать возможности анализа данных.
Описание данных: таблица получена с сайта Открытые данные России
Структура данных:
0 № разрешения
1 Начало срока действия разрешения
2 Окончание срока действия разрешения
3 Организационно- правовая форма владельца
4 Владелец рекламной конструкции
5 Вид рекламной конструкции
6 Размер рекламной конструкции
7 Адрес размещения конструкции
8 Вид собственности недвищимого имущества
9 Основание на установку и эксплуатацию рекламной конструкции (Договор с собственником недвижимого имущества (далее ДСНИ) - для частной собственности; № договора - для муниципальной собственности)
План работы:
Шаг 1. Получение данных и предобработка.
Изменение названий столбцов на более короткие на английском языке. Проверка пропусков, дубликатов и значений данных. Замена типов данных даты и времени. Преобразование столбца "размер рекламной конструкции" в число с вычислением значения площади.
ШАГ 2. Исследовательский анализ.
Найти топ 10 самых активных владельцев рекламных конструкций.
Найти топ 10 улиц, популярных для установки рекламных конструкций.
Определить длительность каждого разрешения и площадь конструкций.
Оценить корреляцию длительности разрешения и площади конструкции. Построить диаграмму рассеяния.
Какие типы конструкций популярны у разных организационно-правовых форм. Построить тепловую карту.
Шаг 3. Статистический анализ.
Проверить гипотезу: средний размер рекламной конструкции различен для частных и муниципальных владельцев.
Шаг 4. Когортный анализ.
Разделить владельцев на когорты по году начала первого разрешения.
Вычислить, сколько новых разрешений получают когорты по годам. Построить тепловую карту.
Шаг 5. Общие выводы.

def area_size(s):
    s_list = s.replace('(', '').replace(')', '').replace(',', '.').split('х')    
    i = 1
    for l in s_list:
        
        if l != '':
            i = i*float(l)
    return i

Шаг 6. Общие выводы.

Наиболее популярные у рекламодателей улицы загружены рекламными конструкциями. Однако действие разрешений ограничено, есть возможность отслеживать освобождающиеся места с помощью открытых данных. Например, многие разрешения на конструкции завершаются в 2023 году. Можно предположить, что это связано с предстоящими выборами Президента в 2024 году. Также в тепловой карте можно обнаружить, что многие разрешения заканчиваются в 2025, 2027 году.

Можно найти наиболее активных владельцев рекламных конструкций (Резник Александр Михайлович, Стрит-Медиа, АТ Меди ...),
И наиболее популярные улицы для установки конструкций (пр. Строителей, ул. Калинина, пр. Победы ... )
Можно описать историю появления действующих на момент публикации данных рекламных конструкций. Владельцы конструкций большинство разрешений получают в первый год своей активности. Высокие показатели активности показывает когорта владельцев 2016 года.

Размер конструкции не имеет связи с длительностью действия разрешения (корреляция 0.72%).
Размер конструкции статистически значимо отличается для частных и муниципальных собственников (средние размеры 17.9, 30.8 соответственно).
